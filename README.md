# MiniMob
A minized mobile rendering game app, cross platform compile.

Language : C++/GLSL  

Current achieved:  
Windows/Android cross platform compile  
Simple mesh rendering  

Next goal:  
Compressed texture support  
Fbx model file support  

Long-range goal:  
PBR Rendering  
Catoon Rendering  
Simple Editor  

Windows Enviroment:
Visual Studio 2019  
Windows SDK 10  
OpenGL ES 2.0  

Android Enviroment:  
Android SDK 29  
Android NDK 22.1.7171670  
OpenGL ES 2.0
